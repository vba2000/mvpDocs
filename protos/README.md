# Protobuf: клиентский контракт WebSocket

## Содержимое каталога

| Файл | Назначение |
|------|------------|
| [ws.proto](ws.proto) | Единый контракт WS: оболочка сервера (`WsEnvelope`), сообщения клиента (`WsClientMessage`), полезные нагрузки в `DataEvent.data` (стакан, сделки, пользовательские события, RFQ и т.д.), коды ошибок |

## Происхождение

Файл **`ws.proto`** скопирован из репозитория бэкенда:

`cex/websocket-gateway/ws-proto/ws.proto`

При обновлении контракта перекопируйте файл в этот каталог и при необходимости обновите страницы в `public/ws/` и `private/ws/`.

## Другие `.proto` в монолите

В сервисах spot/account/market-data/rfq есть **gRPC** и Kafka-прото для внутренней интеграции; для внешних клиентов биржи через WebSocket достаточно **`ws.proto`**.

Файл `ws_kafka_events.proto` в репозитории относится к внутренним событиям Kafka, а не к клиентскому WebSocket API.
