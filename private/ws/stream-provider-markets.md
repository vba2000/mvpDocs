# WebSocket: `provider.markets`

## Имя потока

```
provider.markets
```

Не путать с шаблоном `provider.*` для маркет-мейкерских данных: это **константа** на стороне сервера (`STREAM_PROVIDER_MARKETS`).

## Назначение

После подключения маркет-мейкерской сессии сервер может отправить список доступных для котирования рынков.

## Полезная нагрузка

**`MarketsAvailableProto`**:

| Поле | Описание |
|------|----------|
| `event` | Тип события |
| `markets` | Список идентификаторов рынков (string) |
| `trace_id` | Корреляция |

## Кому доступен

Сессии с ролью **MARKET_MAKER** (отправка привязана к логике `ReactiveEventFanoutService` / handler).
